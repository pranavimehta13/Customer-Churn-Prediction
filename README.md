# 📊 Customer Churn Prediction using ANN

## 📌 Overview

This project focuses on predicting customer churn using an Artificial Neural Network (ANN). The goal is to identify customers who are likely to leave a service, enabling businesses to take proactive retention measures.

---

## 🚀 Features

* End-to-end ML pipeline
* Data preprocessing & feature encoding
* Feature scaling using StandardScaler
* ANN model built using TensorFlow/Keras
* Model evaluation using classification metrics

---

## 📈 Model Details

* Fully connected feedforward neural network
* ReLU activation for hidden layers
* Sigmoid activation for output layer
* Binary cross-entropy loss
* Adam optimizer
